# Security Policy

## Important Notice

This project is **experimental** and has **not been audited for security**.
It is not recommended for production use without thorough review.

## Reporting a Vulnerability

If you discover a security vulnerability, please disclose it responsibly:

1. **Do NOT open a public issue.**
2. Submit your report via [GitHub Security Advisories](https://github.com/aizu-hiroki/spire-plugin-alnumid-credential-composer/security/advisories/new).

Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
